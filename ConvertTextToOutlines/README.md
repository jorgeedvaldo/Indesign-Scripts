# ✏️ ConvertTextToOutlines

This Adobe InDesign script converts all text frames to outlines (vector paths) across all open documents, including master pages. It automatically unlocks all page items before converting, ensuring nothing is missed.

Este script para o Adobe InDesign converte todos os quadros de texto em contornos (caminhos vetoriais) em todos os documentos abertos, incluindo páginas mestras. Ele desbloqueia automaticamente todos os itens da página antes de converter, garantindo que nada seja ignorado.

---

## 📦 Features · Funcionalidades

- 🔓 Automatically unlocks all page items before converting
- 📄 Converts text on all regular pages
- 📐 Converts text on all master pages
- 📂 Processes all open documents at once
- ⚠️ Skips empty text frames and handles errors gracefully

---

## 🖥️ How to Use · Como Usar

1. Open one or more documents in Adobe InDesign.
2. Install the script:
   - Open InDesign and go to `Window > Utilities > Scripts`.
   - In the Scripts panel, right-click on "User" and select `Reveal in Finder` (macOS) or `Reveal in Explorer` (Windows).
   - Copy the file `ConvertTextToOutlines.jsx` into the revealed folder.
3. Back in InDesign, double-click the script to execute it.
4. A dialog will confirm that all documents have been converted successfully.

---

## 📃 Requirements · Requisitos

- Adobe InDesign (CC 2018 or later recommended)
- At least one document must be open

---

## ⚠️ Important Notes · Notas Importantes

- This operation is **irreversible** — text will no longer be editable after conversion. Save a backup of your documents before running this script.
- Esta operação é **irreversível** — o texto deixará de ser editável após a conversão. Guarde uma cópia de segurança dos seus documentos antes de executar este script.
